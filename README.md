简介
reversetclient 程序读取一个文本文件，将其拆分成指定最小和最大长度之间的随机大小的块，并将每个块发送到 reversetserver 程序。服务器反转每个块的内容并将其发送回客户端，客户端将反转的块以倒序保存到输出文件中。
